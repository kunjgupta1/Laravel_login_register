# Laravel_login_register
just try simple login register with laravel
how to use this:

1.crate database
2.GO to .env file and Change database name 
 
3.After  chnages go to your folder and type commands:

    php artisan migrate
    
    php artisan key:generate
    
    php artisan config:cache
    
    php artisan view:cache
    
4.start your browser and go to your public folder and enjoy.
